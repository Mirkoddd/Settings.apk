Ancoraxe da rede USB

O teu ordenador pode compartir a conexión de Internet do teu dispositivo se están ancorados á rede xuntos cun cable USB
  • A ancoraxe á rede funciona con Windows 7, Windows Vista e Linux.

  • Non podes conectar unha tarxeta SD ao ordenador cando está ancorado á rede.

  • Para máis información, incluíndo como usar a ancoraxe á rede do USB con outros sistemas operativos, visita http://www.android.com/tether#usb

Zona interactiva Wi-Fi portátil

Podes converter o teu dispositivo nunha zona interactiva Wi-Fi portátil para compartir a súa conexión a Internet con ordenadores ou outros dispositivos
  • Non se poden utilizar aplicacións do dispositivo para acceder a Internet a través dunha conexión Wi-Fi mentres se utiliza o dispositivo como zona interactiva Wi-Fi portátil

  • Podes configurar a zona interactiva Wi-Fi portátil na configuración da zona interactiva Wi-Fi

  • Se desexas obter máis información, visita http://www.android.com/tether#wifi
