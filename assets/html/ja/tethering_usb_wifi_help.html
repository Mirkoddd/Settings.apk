USBテザリング

USBケーブルでテザリングすると、PCは端末のインターネット接続を共有できます。
  • Windows 7、Windows Vista、Linuxでテザリングします。

  • テザリング中は外部SDカードをPCに接続できません。

  • 他のOSでのUSB接続方法についての詳細情報は http://www.android.com/tether#usb を参照してください。

Wi-Fiテザリング

インターネット接続をPCやその他のデバイスと共有するために、端末でWi-Fiテザリングを使用することができます。
  • 端末をWi-Fiテザリングとして起動中は、Wi-Fi接続でインターネットにアクセスするアプリケーションを利用することはできません。

  • Wi-Fiテザリング設定で、Wi-Fiテザリングを設定できます。

  • 詳細情報は http://www.android.com/tether#wifi を参照してください。
