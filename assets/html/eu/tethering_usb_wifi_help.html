USB ainguraketa

Zure ordenagailuak zure gailuaren internet konexioa parteka dezake, biak USB hariaren bidez lotuz gero
  • Tetheringa egin daiteke Windows 7, Windows Vista eta Linux sistemekin

  • Ezin duzu SD txartelik zure ordenagailura konektatu lotuta bazaude

  • Informazio gehiagorako, bestelako sistema eragileekin lotzeko USBa erabiltzeko modua barne, joan http://www.android.com/tether#usb helbidera

Wi-Fi bero gune ibiltaria

Zure gailua Wi-Fi gune ibiltari bihurtu dezakezu, zure gailuaren internet konexioa gainontzeko ordenagailu nahiz gailuekin partekatzeko
  • Ezin dira gailuaren aplikazioak erabili Internetera Wi-Fi konexio bidez sartzeko gailua Wi-Fi gune ibiltari gisa baliatu bitartean

  • Wi-fi gune ibiltaria konfigura dezakezu Wi-Fi guneen ezarpenetan

  • Informazio gehiagorako, bisitatu http://www.android.com/tether#wifi
